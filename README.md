### Steps to run the project:

#### clone the project:

##### > using SSH: `git clone git@github.com:AimeurAmin/above-episodes-manager.git`

##### > using HTTPS: `https://github.com/AimeurAmin/above-episodes-manager.git`

#### fill in the .env file

#### install libraries:

`npm i`

#### run on dev

`npm run dev`
